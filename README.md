# College Tracker
A console-based application written in **C++** that helps students organize and manage their university application process. Users can keep track of universities, scholarships, application deadlines, and other important information in one place.

## ✨ Features
* Registration and Login System
* Add & manage university applications
* Deadline Tracker with priorities
* Document Manager
* Add & manage scholarship applications

## 🛠 Built With
* C++
* Standard Template Library (STL)
* File I/O

## 🚀 Getting Started
### Requirements
* A C++17 compatible compiler (such as g++)
* Any C++ IDE or editor (Visual Studio Code, Code::Blocks, etc.)

### Running the Project
1. Clone the repository.
2. Compile `main.cpp` using a C++ compiler.
3. Run the executable.

> **Note:** This project stores data in local text files. Sample data files are not included to protect personal information.

## 🔮 Future Improvements & Limitations
* Improve error handling and validation
* Build a graphical user interface (GUI)
* The "Document Management" option on the main screen remains incomplete.
* On the Scholarship page, there is no option to edit or modify previously entered data.
* The Scholarship page offers two options: Edit and View. The Edit option does not immediately return the user to the main screen; one must select the View option first to navigate back to the Home page.
* File saving is restricted to a local computer.
* There is currently no feature to delete user-entered data. 
* Duplicate data entries can occur, leading to data redundancy within the system. 

## 📌 About
This project was initially developed as a school project, but it has the potential to grow into something bigger. It was inspired by my own university application journey and the challenges many students face when organizing applications, scholarships, and deadlines. Through this project, I aimed to build practical C++ programming skills by solving a real-world problem.
